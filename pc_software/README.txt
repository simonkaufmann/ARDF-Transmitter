PC - SOFTWARE:
---

Die PC-Software ist in der Programmiersprache Python verfasst.

Zur Generierung der Exe-Datei wird das Programm PyInstaller benötigt. Die 
Vorgangsweise ist detailliert in der schriftlichen Dokumentation im Abschnitt 
21.4.1 auf Seite 196 beschrieben.

Zum Testen kann das Programm nach Installation des Python-Intepreters 
(https://www.python.org/) Version 2.7 das Programm mithilfe des Befehls "python 
main.py" in der Kommandozeile im Ordner pc_software gestartet werden.
